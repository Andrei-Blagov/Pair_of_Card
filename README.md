 Pair of Card

"Pair of Card" — это небольшая браузерная игра на поиск пар, написанная на чистом JavaScript. Игрок выбирает количество пар (от 2 до 10, только чётные числа) и тип карточек — с числами или изображениями. Карточки переворачиваются, если значения совпадают — остаются открытыми. На прохождение раунда даётся 60 секунд.

## Возможности
- Запуск раунда с выбранным количеством пар (по умолчанию — 4, если введено некорректное значение).
- Два режима: числа или картинки (для картинок используется набор из каталога `img`).
- Визуальная индикация победы при открытии всех пар и сообщения о завершении таймера.
- Простая анимация переворота карточек и подсветка совпавших карт.

## Быстрый старт
1. **Установите зависимости** (требуются только для линтинга кода):
   ```bash
   npm install
   ```
2. **Запустите локальный сервер** — файлы подключаются как ES‑модули, поэтому открывать `index.html` прямо из файловой системы нельзя. Подойдёт любой статический сервер, например:
   ```bash
   npx http-server . -c-1
   # или
   npx serve .
   ```
3. Откройте в браузере адрес, который напечатает сервер (обычно `http://localhost:8080` или `http://localhost:3000`).
4. Нажмите кнопку «Начать игру» и следуйте подсказкам всплывающих окон.

## Как играть
1. После нажатия «Начать игру» введите чётное количество пар от 2 до 10. Если введено неверно, автоматически будет выбрано значение 4.
2. Укажите режим: «картинки» или «числа».
3. Открывайте по две карточки. Если значения совпали — пара остаётся открытой. Если нет — карточки перевернутся обратно через секунду.
4. Раунд завершается победой при открытии всех пар или поражением после истечения 60 секунд. После завершения страница перезагрузится через 5 секунд, чтобы можно было начать заново.

## Скрипты
- `npm run lint` — проверка кода ESLint (конфигурация Airbnb Base).

## Структура проекта
- `index.html` — точка входа, подключает стили и главный скрипт.
- `main.js` — создание игрового поля, логика таймера и выбор режима.
- `card.js` — базовый класс карточки с числовым отображением.
- `amazingCard.js` — расширение карточки с выводом картинок.
- `style.css` — оформление и анимации карточек, сообщений и кнопок.
- `img/` — изображения для лицевой стороны карт и фоновых элементов.

## Полезные советы
- Если картинки не загрузились, в консоли будет предупреждение — используется запасное изображение `img/default.jpg`.
- Для удобства разработки можно использовать расширение Live Server в VS Code или любой другой статический сервер.
